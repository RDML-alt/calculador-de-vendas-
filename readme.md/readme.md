# Calculadora de Desconto

Projeto desenvolvido para aplicar regras de negócio e testes unitários.

## Estrutura do Projeto
src → código fonte
tests → testes unitários
docs → documentação

## Regras
- Valor < 100 → sem desconto
- Valor entre 100 e 500 → 10%
- Valor > 500 → 20%
- Cliente VIP → +5%
- Valor não pode ser negativo

## Executar projeto
npm install
npm start

## Executar testes
npm test
